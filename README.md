A live football scores platform built with **Node.js**, **NestJS**, **MongoDB**, and **Socket.io**. This project provides real-time updates for matches, players, teams, referees, and leagues.

---

## Features

- Live match updates with **real-time scores** using **Socket.io**.
- Player, referee, team, stadium, and league management.
- Support for different tournament types: **league** and **knockout**.
- Fully RESTful API endpoints for CRUD operations.
- Admin panel ready for managing data (future integration).
- Structured and scalable backend architecture using **NestJS**.

---

## Tech Stack

- **Backend:** Node.js, NestJS, Express
- **Database:** MongoDB & Mongoose
- **Realtime Updates:** Socket.io
- **Version Control:** Git & GitHub
